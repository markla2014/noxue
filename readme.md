### 开发规范

1. 所有文件小写字母组成，或下划线分隔
1. 方法名 采用pascal命名法，格式 类名+操作，例如：UserInsert，ArticleSelect
1. 局部变量采用骆驼命名法，全局变量采用pascal命名法
